# Notes

## Simulator

We use the *stage_ros* ROS package as a wrapper for the Stage 2D simulator

## Navigation

We use ROS *move_base* with the *teb_local_planner*

## Usage

```
roslaunch soccer_stage soccer_world.launch
```
